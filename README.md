# Сайт о творчестве Винсента Ван Гога

Простой статический сайт, демонстрирующий основные картины Винсента Ван Гога.

## Что включает

* Главная страница с общей информацией
* Галерея картин в три колонки
* Раздел об художнике
* Форма обратной связи


## Технологии

* HTML5 — структура страниц
* CSS3 — стилизация и адаптивный дизайн
* Grid Layout — расположение элементов в галерее

## Как запустить локально

1. Скачайте репозиторий
2. Откройте файл `index.html` в браузере
